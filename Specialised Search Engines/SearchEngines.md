# Specialized Search Engines
### Shodan
```
https://www.shodan.io/
```
It allows you to search for specific types and versions of servers, networking equipment, industrial control systems, and IoT devices.
### Censys
```
https://search.censys.io/
```
It focuses on Internet-connected hosts, websites, certificates, and other Internet assets. 
### VirusTotal
```
https://www.virustotal.com/gui/home/upload
```
It an online website that provides a virus-scanning service for files using multiple antivirus engines.  
### Have I Been Pwned (HIBP) 
```
https://haveibeenpwned.com/
```
It tells you if an email address has appeared in a leaked data breach.
